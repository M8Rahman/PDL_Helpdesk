## Font Files Directory

Place your Spiffy Script SSi font files here:

- SpiffyScriptSSi.woff2   ← preferred (smallest, fastest)
- SpiffyScriptSSi.woff    ← fallback for older browsers
- SpiffyScriptSSi.ttf     ← fallback if woff not available

The CSS in static/css/fonts.css will load them automatically.

If you only have a .ttf file, convert it to .woff2 using:
https://cloudconvert.com/ttf-to-woff2

Until the font file is added, the system uses "Dancing Script" (Google Fonts)
as a graceful elegant fallback.
